# Quant_Lab — Python 量化交易互動教學平台

![Quant_Lab Banner](https://img.shields.io/badge/Status-Completed-success?style=for-the-badge&logo=react) ![Pyodide](https://img.shields.io/badge/Powered%20by-Pyodide-blue?style=for-the-badge&logo=python) ![PWA](https://img.shields.io/badge/PWA-Ready-f59e0b?style=for-the-badge&logo=pwa)

**Quant_Lab** 是一個基於 **React**、**TypeScript**、**Vite** 與 **Pyodide** 開發的「全前端」Python 量化交易互動教學平台。

你可以直接在瀏覽器中學習理論、編寫、調整參數，並即時執行 Python 量化策略代碼，體驗專業的回測視覺化輸出。完全**無需安裝任何伺服器**或本地環境！

---

## ✨ 平台亮點與技術棧

- **0 伺服器依賴**: 透過 WebAssembly (Pyodide) 直接在前端執行 Python 邏輯、NumPy 運算與完整的事件驅動回測引擎。
- **現代化原子設計 UI**: 
  - 支援一鍵切換**亮色 (Light) / 暗色 (Dark) 高對比主題**。
  - 專業的**終端機 (Terminal)** 模擬輸出與 **CodeMirror** 程式碼編輯器。
  - **自動儲存與同步**: 編輯器內容自動儲存至 localStorage，不怕遺失草稿。
  - **可調式面板**: 支援拖拉縮放，自由分配理論與程式開發空間。
- **豐富的視覺化圖表**: 
  - 使用 Chart.js 渲染多功能圖表，支援**十字線同步 (Crosshair Sync)**。
  - 提供**成交量 (Volume)** 與進階 **K 線圖 (Candlestick)** 渲染。
  - **SVG 動畫教學**: 核心理論配有動態路徑動畫，強化學習感知。
- **PWA 支援**: 可作為獨立應用安裝至桌面 (Windows/Mac) 或手機，並支援離線快取，隨時隨地學習量化。

## 📚 漸進式量化學習路徑 (持續擴充中)

平台精心設計了循序漸進的模組，總計 25+ 個經典必學案例。帶你從最基礎的均線指標，走向進階動態突破與資金管理。

### 🔰 模組一：量化入門 (奠定基礎)
- **[1.1] 雙均線策略**: 認識量化交易。學習短均線與長均線的金叉與死叉買賣訊號，以及基礎的回測引擎觀念。

### 📈 模組二：趨勢跟蹤 (過濾與自適應)
- **[2.1] 經典 MACD 策略**: 利用平滑異同移動平均線的零軸穿越來捕捉市場趨勢。
- **[2.2] ADX 輔助 MACD**: 導入趨勢強度指標 (ADX)，過濾無方向的震盪市假訊號。
- **[2.3] 自適應均線 (AMA)**: 根據市場當下波動率，自動動態調整平滑係數的均線系統。
- **[2.4] 阿隆指標 (AROON)**: 從「距離上一次創新高的時間」來捕捉趨勢發韌初期的領先指標。
- **[2.5] 簡易波動 EMV**: 將成交量 (Volume) 與價格波幅結合，找出阻力最小的動能趨勢。
- **[2.6] 布林帶 (Bollinger Bands) [NEW]**: 經典的波動率通道策略，探討標準差與均值回歸。
- **[2.7] RSI 超買超賣策略 [NEW]**: 利用強弱指標捕捉市場極端情緒與反轉時機。
- **[2.8] KDJ 隨機指標 [NEW]**: 結合動能與超買超賣的多層次判斷。

### ⚡ 模組三：突破策略 (尋找壓力與支撐)
- **[3.1] Dual Thrust 區間突破**: 根據前 N 日震幅動態推算今日的開盤上下軌操作區間。
- **[3.2] 日內高低點突破**: 捕捉開盤初期的劇烈市場情緒方向。
- **[3.3] 增強版唐奇安通道**: 傳奇「海龜交易法則」的核心，具備動態停利止損線。
- **[3.4] Hans123 日內突破**: 外匯市場常用的開盤時段標竿突破策略。
- **[3.5] 菲阿里四價**: 日本傳奇操盤手的經典防線策略。
- **[3.6] 動態波幅突破**: 利用時間序列的波動率 (ATR) 自動伸縮突破通道。
- **[3.7] R-breaker 交易策略**: 結合「趨勢突破」與「逆勢均值回歸」的高階策略。

### 🔄 模組四：切換策略 (應對不同市況)
- **[4.1] 經典恆溫器策略**: 使用 CMI 指標區分趨勢/震盪市場，自動切換邏輯核心。
- **[4.2] MACD + RSI 雙重過濾 [NEW]**: 示範多指標共振，提升勝率的實戰技巧。
- **[4.3] 趨勢/震盪自動切換器 [NEW]**: 進階版自適應系統，動態分配策略權重。

### ⚖️ 模組五：高級交易與套利 (擺脫大盤單邊風險)
- **[5.1] 期貨跨期套利實戰**: 尋找高度相關資產的「價差擴張與收斂」。
- **[5.2] 乖離率 BIAS 策略**: 尋找物極必反的均值回歸點。

### 🛡️ 模組六：風險與資金管理 (生存的關鍵)
- **[6.1] 馬丁格爾策略與倍增法**: 探討賭徒謬誤與逆勢加倉的危險。
- **[6.2] 凱利公式部位管理**: 計算最佳下注比例，優化長期資產增長。
- **[6.3] 反向馬丁格爾 (逆勢倍增法)**: 「截斷虧損，讓獲利奔跑」的聖杯思維。

### 🤖 模組八：機器學習量化入門 [NEW]
- **[8.1] 線性迴歸預測**: 使用數學回歸預測短期價格走勢。
- **[8.2] 動量因子策略**: 因子投資入門，尋找持續性表現的 alpha。
- **[8.3] 特徵工程展示**: 如何將多個技術指標轉化為機器學習的輸入特徵。

### 📊 模組九：實戰進階 [NEW]
- **[9-1] 多股票指數輪動**: 實作動態權重分配與強勢股輪動邏輯。
- **[9-2] 蒙地卡羅壓力測試**: 使用隨機模擬驗證策略的健壯性。

## 🕹️ 快速開始

1. 從左側導覽列選擇單元。
2. 閱讀**核心理論**並觀察**動態 SVG 動畫**。
3. 調整右側**參數滑桿**觀察代碼變化。
4. 點擊 **「▶ Run」** 即時執行 Python 回測。
5. 在 **Result** 查看自動同步的十字線圖表與績效報告。

*(註：本互動教學平台僅供學習與交易邏輯研究使用，請勿直接用於實盤自動交易。)*
